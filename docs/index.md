# Storm Council Wiki

Storm Council is a contradiction-aware research workflow for turning a contested
question into source-mapped claims, an explicit contradiction ledger, a decision
brief, and a self-contained HTML report.

Use this local wiki when reading the project documentation is easier than
jumping between Markdown files in the repository.

## Start here

- [Methodology](methodology.md) explains the six-stage workflow and why Council
  Mode is more than parallel persona answers.
- [Worked examples](examples.md) points to committed runs and the recommended
  reading order.
- [Claim traceability](claim-traceability.md) defines claim, source, evidence,
  contradiction, and verification fields.
- [Safety and limitations](safety-and-limitations.md) describes what Storm
  Council supports and what still needs human or domain-expert review.
